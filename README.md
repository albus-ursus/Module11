# Задание Module №11

____

## Пошаговое описание действий:

- [ ] **Создайте новый проект и подключите в него Git через терминал.**
    - Создание папки с проектом.
    - Создание файла ***index.html***.
    - Создание папки ***css***.
    - Создание файла ***style.css*** в папке ***css***.
    - Выполнение команды **git init** *- инициализирует (т.е. cоздает) репозиторий*.
- [X] Шаг выполнен.

____

- [ ] **Укажите, что в вашем проекте будут игнорироваться все файлы с расширением .sass и папки: bin, admin, config.**
    - Создание файла ***.gitignore*** в папке с проектом.
    - Внесение в файл ***.gitignore*** следующих параметров: **.sass*, */bin*, */admin*, */config*.
- [X] Шаг выполнен.

____

- [ ] **Добавьте все файлы в локальное хранилище.**
    - Выполнение команды **git add .** *- добавляет изменение из рабочего каталога в раздел проиндексированных файлов*.
    - Выполнение команды **git commit -m 'first commit'** *- запись индексированных изменений в репозиторий Git*.
- [X] Шаг выполнен.

____

- [ ] **Создайте новую ветку и перейдите в неё.**
    - Выполнение команды **git branch module11** *- создание ветки "module11"*.
    - Выполнение команды **git checkout module11** *- переход на ветку "module11"*.
- [X] Шаг выполнен.

____

- [ ] **В новой ветке создайте папку blog с файлами: index.js, index.html.**
    - Создание папки ***blog***.
    - Создание файлов ***index.js, index.html*** в папке ***blog***.
- [X] Шаг выполнен.

____

- [ ] **Загрузите на удаленный репозиторий лишь основную ветку вашего проекта.**
    - Выполнение команды **git add .** *- добавляет изменение из ветки "module11" в раздел проиндексированных файлов*.
    - Выполнение команды **git commit -m 'first commit module11'** *- запись индексированных изменений ветки "module11" в репозиторий Git*.
    - Выполнение команды **git checkout master** *- переход на основную ветку*.
    - Создание ***README.md*** в папке проекта.
    - Выполнение команды **git remote add origin https://github.com/albus-ursus/Module11.git** *- создание нового подключения к удаленному репозиторию с именем origin*.
    - Выполнение команды **git push -u origin master** *- публикация основной ветки в удаленном репозитории*.
- [X] Шаг выполнен.
